Interface hermpic
=================

1. Versionsinformationen

  * \HPrevision -> Version des Macropacketes

  * \HPdescription -> Kurzbeschreibung

  * \HPlongrevision -> Lange Versionsbeschreibung

  * \HPdate -> Datum der letzten Änderung
 
2. Parameter

  * verbatim -> viel Ausgaben
     -> besser waehre nur logfile (OK)

  * center -> zentrieren der Schemata (OK)

  * box -> umranden der Schemata (OK)

  * rel_0_6 -> waehlt Inteface von Version 0_6 (OK)
     => in 1.0 nicht mehr enthalten

3. Schema

  * \schema -> (x,y) (OK)

  * \entity (x,y) {name} (OK)

  * \relation (x,y) {name} (OK)

  * \cluster (x,y) {name} (OK)

  * \attr (OK)

  * \key {name} (OK)

  * \connection (OK)
    -> \conn als Abkürzung

4. Integritaetsbedingungen

  * \incude, \exclude, \disjunct
